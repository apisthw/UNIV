# เข็มทิศ (Kem Tid)

เว็บแอปฝึกและติดตามการเตรียมสอบเข้า **คณะวิศวกรรมศาสตร์** ในระบบ TCAS ของไทย เจาะกลุ่มสายวิศวกรรมไฟฟ้า / คอมพิวเตอร์ / ปัญญาประดิษฐ์ (AI) โดยเฉพาะ เป้าหมายหลักคือ 3 พระจอมฯ (มจธ., สจล., มจพ.) และมหาวิทยาลัยอื่นที่มีคณะวิศวะ (จุฬาฯ, ธรรมศาสตร์, เกษตรศาสตร์, มหิดล, ศิลปากร, เชียงใหม่, ขอนแก่น, สงขลานครินทร์, สุรนารี)

**ใช้งานได้ที่**: https://apisthw.github.io/UNIV/

## Features

| แท็บ | หน้าที่ |
|---|---|
| **ภาพรวม** | สรุปสถิติผู้เรียน, การ์ด "ทำต่อจากจุดที่ค้างไว้", กำหนดการ TCAS, ประกาศข่าว |
| **ปฏิทิน & เกณฑ์** | เส้นเวลา TCAS70, ศูนย์ลิงก์ทางการ (ตรวจสอบแล้ว) |
| **คลังข้อสอบ** | แบบฝึกหัดพร้อมคำใบ้ เฉลยทีละขั้น และผังหัวข้อ (Coverage Map) — 8 วิชา (TPAT3, A-Level คณิต 1/2, ฟิสิกส์, เคมี, อังกฤษ, TGAT, เสริมความรู้ EE/CpE) |
| **บทเรียน** | สรุปเนื้อหา+สูตรต่อหัวข้อ พร้อม checklist ติดตามความคืบหน้า |
| **คะแนนวิศวะ** | คะแนนสูงสุด–ต่ำสุด TCAS69 รอบ 3 เทียบระหว่างมหาวิทยาลัย พร้อมเครื่องคำนวณโอกาสจากคะแนนของตัวเอง |
| **ความคืบหน้า** | มิเตอร์ความพร้อมต่อวิชา, จุดอ่อนที่ควรทบทวน |

## สถาปัตยกรรม

- **ไม่มี build step** — `index.html` เปิดตรงได้ทั้งแบบ double-click และผ่านเว็บ (GitHub Pages)
- Vanilla JavaScript, Tailwind CSS ผ่าน CDN, ไม่มี framework/backend
- ข้อมูลวิชาแยกไฟล์ใน `data/*.js` โหลดผ่าน `<script src>` (ไม่ใช่ `fetch()` เพื่อให้เปิดจาก `file://` ได้ด้วย)
- บันทึกความคืบหน้าใน `localStorage` ของเบราว์เซอร์ — ไม่มี account, ไม่ sync ข้ามอุปกรณ์

```
├── index.html       ← HTML + CSS + JS logic ทั้งหมด
└── data/
    ├── tpat3.js
    ├── alevel-math1.js
    ├── alevel-phys.js
    ├── alevel-chem.js
    ├── alevel-math2.js
    ├── alevel-eng.js
    ├── tgat.js
    ├── enrichment.js
    └── tcas-scores.js
```

## ข้อควรรู้

- ข้อสอบในคลังข้อสอบเป็น **แบบฝึกหัดที่เขียนขึ้นใหม่** อ้างอิงสไตล์จากสถาบันติวที่มีชื่อเสียง ไม่ใช่ข้อสอบจริงที่หลุดออกมา
- หัวข้อ **มิติสัมพันธ์** (Diagrammatic/Spatial Reasoning) ของ TPAT3/TGAT2 ไม่มีชุดฝึกในตัวเว็บ เพราะต้องใช้สื่อภาพที่ตรวจสอบด้วยสายตา — มีลิงก์ไปแหล่งฝึกจริง (mytcas, SmartMathPro) แทน พร้อมติ๊กบันทึกความคืบหน้าได้ในแท็บบทเรียน
- ตารางเปรียบเทียบเกณฑ์คะแนนบางส่วน (`compareData`) ยังเป็นข้อมูลตัวอย่าง มีแบนเนอร์เตือนในหน้าเว็บ — กำหนดการสอบและคะแนน TCAS69 เป็นข้อมูลค้นคว้าและตรวจสอบไขว้จริงแล้ว
- A-Level ทักษะฟัง-พูดยังไม่มี เพราะต้องใช้ไฟล์เสียงจริง
